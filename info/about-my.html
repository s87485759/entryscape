My Confolio

my.confolio.org is a generic test installation of the Confolio system where anyone that asks nicely can have an account for experimenting with the system. Unfortunately, no guarantees can be given at this time regarding uptime, stability or continuity so don't rely on this system for any critical activities.

Confolio

Confolio is an electronic portfolio system based on the resource and metadata management framework SCAM.
It provides a wide range of functionality, such as:

  • Management and advanced description of resources such as uploaded files, links, and non-digital objects
  • Collaborative creation and annotation of resource
  • Integration of other online services such as Flickr, YouTube, Google Calendar, etc
