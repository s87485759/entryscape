The Different Views in EntryScape

  • start - a nice overview for newcomers and experts
    • Welcome message
    • Easy login and signup buttons
    • A searchable list of users, groups, and portfolios
    • A list or recent updated material from the entire system (excluding internal references)
  • signup - give the user a chance to create an account and a corresponding portfolio
    • display and require acceptance of a terms and conditions
    • form for filling in minimal user information
    • account name and password
  • default - for working or inspecting folders and their content
    • Contents of a folder as a list or as icons with titles
    • Folder path where the current folder is contained
    • A folder tree for the current portfolio is displayed on demand
    • User can further inspect the folder or one of its children by selecting it
    • User can open subfolders or open content that has digital representations
    • User can download content that has digital representations
    • Preview of user selected content if possible (e.g. embedded youtube)
    • If no preview an appropriate icon is shown
    • Information about the selected content
  • user profile - a nice overview of a user
    • Profile picture if it exists
    • User profile information
    • Connection to home portfolio
    • Membership in groups
    • Portfolios and folders where the user has explicit write access
    • A list of recent created or modified content by this user (references are excluded)
  • group profile - a nice overview of a group
    • Group picture if it exists
    • Group profile information
    • Members of the group
    • Portfolios and folders where the group has explicit write access
    • A list of recent created or modified content in portfolios where this group has read access and all entries that are explicitly readable by the group (references are excluded)
  • search - a view showing a search form with many options on the side of the search results
    • launched from search field in the navigationbar
    • list of search results
    • free-text field
    • choose between searching in all portfolios, user portfolio (if logged in), or specific portfolio
    • choose between sorting by title or modification date, possibly in reverse order
    • restrict to specific link types (advanced)
    • restrict to specific mime-types (advanced) TODO
    • restrict to specific resource type
  • help - a view which shows help information
  • about - a view showing about information
